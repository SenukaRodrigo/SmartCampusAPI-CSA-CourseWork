# Smart Campus Sensor & Room Management API

## Overview
RESTful API developed for 5COSC022W Client-Server Architectures coursework using Java, Maven, and JAX-RS (Jersey).

The system manages:

- Rooms
- Sensors
- Sensor readings/history

## Technology Stack

- Java 21
- Maven
- JAX-RS (Jersey)
- Grizzly Embedded Server
- HashMap / ArrayList (No database)

## Run the Project

```bash
mvn compile
mvn exec:java
